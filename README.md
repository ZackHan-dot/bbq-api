## Description

这是NestJs项目的模板，包含基础的认证和鉴权功能，统一的异常处理器、响应格式，基础的日志服务，且包含一些基于RBAC模型的CURD资源等。

## Project setup

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
